### 🚧 Development Roadmap
- [x] **Core API** – FastAPI + Swagger (v1.0.0)
- [x] **AI Assistant** – Gemini intent extraction + ISO 8601 normalization
- [x] **Database** – PostgreSQL + SQLAlchemy with strict constraints
